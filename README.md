Question n°1
Par qui, quand, pourquoi et où a été inventé Internet ?
Internet a d'abord été invité pour l'armée américaine, dans un cadre de projet du département de la défense pour facilité la détection des problèmes ou même détecter des missiles nucléaires, il a été créé par J.C.R. Licklider, Léonard Kleinrock et Robet Taylor. Internet a été créé en Amérique.

Question n°2
Quelle est la différence entre le Web et Internet ?
Internet (pour Inter Network) est une infrastructures qui relie tout les ordinateurs dans le monde entier par un principe d'IP (Internet Protocol), c'est un réseau se composant de millions de réseaux publiques alors que le Web (ou World Wide Web) est un additionnel à Internet, le web permet le partage d'information par Internet.
